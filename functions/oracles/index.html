Goal-oriented Oracles

Glycogen Synthase Kinase 3 Beta (GSK3β)

Description: Glycogen synthase kinase 3 beta, also known as GSK3β, is an enzyme that in humans is encoded by the GSK3β gene. Abnormal regulation and expression of GSK3β is associated with an increased susceptibility towards bipolar disorder. The oracle is a random forest classifer using ECFP6 fingerprints using ExCAPE-DB dataset. 

from tdc import Oracle
oracle = Oracle(name = 'GSK3B')
smiles_lst = ['CC(C)(C)....' 'C[C@@H]1....','CCNC(=O)....', 'C[C@@H]1....']
oracle(smiles_lst)
# [0.03, 0.02, 0.0, 0.1]
oracle('CC(C)(C)....')
# 0.03

References:

[1] Li, Yibo, Liangren Zhang, and Zhenming Liu. “Multi-objective de novo drug design with conditional graph generative model.” Journal of cheminformatics 10.1 (2018): 33.

[2] Jin, Wengong, Regina Barzilay, and Tommi Jaakkola. “Multi-objective molecule generation using interpretable substructures.” ICLR. 2020.

[3] Sun, Jiangming, et al. “ExCAPE-DB: an integrated large scale dataset facilitating Big Data analysis in chemogenomics.” Journal of cheminformatics 9.1 (2017): 17.

IBM RXN Synthetic Accessibility (IBM_RXN)

Description: Gao and Coley [1] have demonstarted that surrogate scoring models cannot well determine the easiness to obtain a chemical and therefore, in addition to the SA oracle, we provide a score generated by full retrosynthetic pathway analysis. TDC included interfaces for multiple types of retrosynthetic pathway analysis as oracles and provided flexible access to various information of the result. For the sake of handiness of TDC and IP of the retrosynthetic analysis software, we utilize the API access of those software and require additional input to the oracle function. Current release only include IBM_RXN with more coming. 

from tdc import Oracle
oracle = Oracle(name = 'IBM_RXN')
smiles = 'CCOCCOCC'
key = 'apk-c9db......' # You can obtain a key from https://rxn.res.ibm.com
oracle(smiles, key)
# 0.983

Rediscovery

Description: This oracle aims to rediscover the target molecule Celecoxib, Troglitazone, and Thiothixene. Specifically, it aims for the generated molecule to have high tanimoto similarity with Celecoxib. From Guacamol Benchmark.

from tdc import Oracle
oracle = Oracle(name = 'Rediscovery')
smiles_lst = ['CC(C)(C)....' 'C[C@@H]1....','CCNC(=O)....', 'C[C@@H]1....']
oracle(smiles_lst)
# {'Celecoxib': [0.03, 0.32, 0.01, 0.1], 'Troglitazone': [0.33, 0.12, 0.0, 0.12], 'Thiothixene': [0.032, 0.021, 0.02, 0.1]}
oracle('CC(C)(C)....')
# {'Celecoxib': 0.03, 'Troglitazone': 0.01, 'Thiothixene': 0.6}

Note: You can also access individual oracle in the set. For example, 

from tdc import Oracle
oracle = Oracle(name = 'Celecoxib_Rediscovery')
smiles_lst = ['CC(C)(C)....' 'C[C@@H]1....','CCNC(=O)....', 'C[C@@H]1....']
oracle(smiles_lst)
# [0.03, 0.32, 0.01, 0.1]

References:

Brown, Nathan, et al. “GuacaMol: benchmarking models for de novo molecular design.” Journal of chemical information and modeling 59.3 (2019): 1096-1108.

Distribution Learning Oracles

Diversity

Description: The diversity of a set of molecules is defined as the average pairwise Tanimoto distance between the Morgan fingerprints.

from tdc import Oracle
oracle = Oracle(name = 'Diversity')
smiles_lst = ['CC(C)(C)....' 'C[C@@H]1....','CCNC(=O)....', 'C[C@@H]1....']
oracle(smiles_lst)
# 0.2

References:

Benhenda, Mostapha. “ChemGAN challenge for drug discovery: can AI reproduce natural chemical diversity?.” arXiv preprint arXiv:1708.08227 (2017).

KL divergence

Description: KL divergence between the probability distributions of a variety of physicochemical descriptors for the training set and a set of generated molecules. Models able to capture the distributions of molecules in the training set will lead to small KL divergence values. To increase diversity, we want high KL.

from tdc import Oracle
oracle = Oracle(name = 'KL_Divergence')
gen_smiles_lst = ['CC(C)(C)....' 'C[C@@H]1....','CCNC(=O)....', 'C[C@@H]1....']
train_smiles_lst = ['CCNC[C@@H]....', '[C@H](O)C2)....', 'C(=O)CCCc2....', 'sc(NC(=O)....']
oracle(gen_smiles_lst, train_smiles_lst)
# 0.2

References:

Brown, Nathan, et al. “GuacaMol: benchmarking models for de novo molecular design.” Journal of chemical information and modeling 59.3 (2019): 1096-1108.

Frechet ChemNet Distance (FCD)

Description: FCD first takes the means and covariances of the activations of the penultimate layer of ChemNet are calculated for the reference set and for the set of generated molecules. The FCD is then calculated as the Frechet distance for both pairs of values. Similar molecule distributions are characterized by low FCD values.

from tdc import Oracle
oracle = Oracle(name = 'FCD_Distance')
smiles_lst = ['CC(C)(C)....' 'C[C@@H]1....','CCNC(=O)....', 'C[C@@H]1....']
ref_smiles_lst = ['CCNC[C@@H]....', '[C@H](O)C2)....', 'C(=O)CCCc2....', 'sc(NC(=O)....']
oracle(smiles_lst, ref_smiles_lst)
# 0.2

References:

Brown, Nathan, et al. “GuacaMol: benchmarking models for de novo molecular design.” Journal of chemical information and modeling 59.3 (2019): 1096-1108.
