Caco-2

Dataset Description: The Caco-2 cell effective permeability (Peff) is an in vitro approximation of the rate at which the drug passes through intestinal tissue.

Task Description: Given a drug SMILES string, predict the Caco-2 cell effective permeability.

Dataset Statistics: 910 drugs.

Dataset Split: Random split in default. Recommend scaffold split. 

from tdc.single_pred import ADME
data = ADME(name = 'Caco2_Wang')
split = data.get_split()

References:

[1] Ning-Ning Wang et al, ADME Properties Evaluation in Drug Discovery: Prediction of Caco-2 Cell Permeability Using a Combination of NSGA-II and Boosting, Journal of Chemical Information and Modeling 2016 56 (4), 763-773
