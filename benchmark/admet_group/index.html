ADMET Benchmark Group

ADMET is one of the cornerstones for small molecule drug discovery, where it defines the drug's efficacy and toxicity's profile. Thus, a machine learning model that can accurately predict all of the ADMET properties given the drug's structural information is highly valuable. We select 18 ADMET datasets from TDC's collection and formulate it as a benchmark group. To access, type: 

from tdc import BenchmarkGroup
group = BenchmarkGroup(name = 'ADMET_Group', path = 'data/')

Follow the instruction on how to use the BenchmarkGroup class. For every dataset, we use scaffold split with 70%/10%/20% training/validation/testing fractions.
