Drug Synergy Benchmark Group

Drug combination therapy is used to treat many diseases, such as almost all cancer chemotherapy and many antimicrobial regimens, or preventing cancer. They also allow lower doses of each drug and consequently lower adverse effects. Thus, accurately predicting the synergy of two drugs in treating a disease is of tremendous importance. In this benchmark group, we use the DrugComb dataset. We provide five synergy scores to predict, along with the evaluation across tissue types during testing. To access, type: 

from tdc import BenchmarkGroup
group = BenchmarkGroup(name = 'DrugSyn_Group', path = 'data/', file_format='pkl')

Follow the instruction on how to use the BenchmarkGroup class. For every dataset, we use cold drug combination split into 70%/10%/20% training/validation/testing fractions. The evaluation metrics is MAE.

We encourage submissions that reports results for all five synergy scores. Note that the tissue types are automatically calculated based on the test set prediction on CSS data type.
