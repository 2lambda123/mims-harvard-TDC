USPTO

Dataset Description: The full USPTO (United States Patent and Trademark Office) retrosynthesis dataset.

Task Description: Given reactant set X, predict the reaction product Y.

Dataset Statistics: 1,939,253 reactions.

Dataset Split: Random Split 

from tdc.generation import Reaction
data = Reaction(name = 'USPTO')
split = data.get_split()

References:

[1] Daniel Lowe. Chemical reactions from US patents (1976-Sep2016).

[2] Jin, Wengong, et al. “Predicting organic reaction outcomes with weisfeiler-lehman network.” NeurIPS. 2017.
