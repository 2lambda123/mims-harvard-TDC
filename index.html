Therapeutics Data Commons

Datasets for Medicinal Machine Learning

Getting Started

What is TDC?

An open, user-friendly and extensive data hub that curates 50+ machine learning-ready datasets across 20+ therapeutics tasks, ranging from target discovery, efficacy, safety, manufacturing to clinical trials for both small molecule and biologics.

Easy to Install and Use

Access to ML-ready data in 3 lines of codes. TDC also provides numerours helper functions such as to PyG/DGL graph, label distribution viz and etc.

Extensive Data and Task Coverage

TDC covers a wide range of established and emerging therpeutics tasks. Datasets are curated to have the highest standard. TDC also provides a benchmark and leaderboard.

Community-Driven Effort

Extremely easy to contribute your dataset to TDC. Just prepare a csv file and it will be shared to the TDC community.
