TDC Dataset Overview

TDC is a data hub for machine learning for therapeutics research.

  • From Bench to Bedside: TDC strives to cover most machine learning tasks in the therapeutics pipelines, from wet-lab target identification, drug discovery, all the way to manufacturing.
  • Keep Abreast with the Forefront: TDC is updated regularly to reflect the most recent innovation. It currently extends above small-molecule drugs and cover important tasks in biologics development.
  • Machine Learning Ready: TDC provides input features for various biomedical entities in the dataset. These features are carefully selected based on their usage in current published machine learning models.
TDC is an open-source effort. We welcome contributions from the research community. If you want to get involved, please fill out this form form!

Tasks in TDC

To give a hoslitic view of the therapeutics landscape TDC covers, we categorizes each therapeutic task in four therapeutics areas (target discovery, activity, efficacy & safety, and manufacturing) across the two therapeutics products (small molecule drugs and biologics), given different machine learning problems. For each (problem, area, product type), we identify a number of prediction tasks and associated datasets. Click each task tag to explore its datasets.
